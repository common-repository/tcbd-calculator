== Description ==

This plugin will enable awesome calculator in your Wordpress theme.


= Plugin Features =

* Only 5 KB.
* Easy to use.
* One click to install
* MCE Button On editor

= Shortcode =
`[tcbd-calculator]`


== Screenshots ==

1. TCBD Calculator Plugin preview.
2. TCBD Calculator Plugin shorcode generator.

== Installation ==

= Installation =

1. Upload 'tcbd-calculator' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
